# 📝 Gerenciador de Notas (Vanilla JS)

Este é um sistema de CRUD (Create, Read, Update, Delete) desenvolvido inteiramente com **JavaScript Puro**, focado em manipulação de arrays e estruturas de controle.

## 🚀 Sobre o Projeto
O projeto foi criado para consolidar conhecimentos fundamentais de programação, como funções, arrays, objetos e loops. 
Ele utiliza uma interface baseada em comandos via `prompt` e `alert`, simulando a lógica de um sistema de gestão escolar diretamente no navegador.

## ✨ Funcionalidades
- **Adicionar Nota**: Insere uma nova nota ao final da lista (`push`).
- **Listar Notas**: Exibe todas as notas cadastradas formatadas (`join`).
- **Atualizar Nota**: Altera um valor específico baseado no seu índice.
- **Remover Registros**: Opções para remover a primeira (`shift`) ou a última nota (`pop`) da lista.
- **Validação**: Verifica se existem notas antes de tentar remover ou se o índice informado é válido.

## 🛠️ Tecnologias e Conceitos
- **JavaScript (ES6+)**: Uso de funções, arrays e template strings.
- **Estruturas de Repetição**: `do...while` para manter o menu ativo.
- **Estruturas de Controle**: `switch/case` e `if/else` para lógica de decisão.
- **Manipulação de Arrays**: Prática com os métodos `.push()`, `.pop()`, `.shift()` e `.join()`.

Desenvolvido como parte dos estudos de Front-End.
